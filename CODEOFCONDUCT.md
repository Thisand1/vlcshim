# Code of Conduct

This project has a blunt README and a strong personality, but the **Code of Conduct** below sets the baseline of expected behavior when interacting with this repository, its issues, and discussions.

We maintain this project in a community-focused way. By participating in this project, you agree to respect the following guidelines:

---

## The Standards In This Project

Be respectful.  
Be considerate.  
Be inclusive.  
Communicate clearly.  
Don’t harass people or attack someone personally for who they are.

Specifically, avoid:

- Personal attacks or insults targeting intrinsic traits (age, race, gender, identity, disability, etc.)
- Harassing language that targets a person rather than a behavior
- Inappropriate content in issues, comments, or pull requests
- Dismissing questions just because they seem “basic”

If you disagree with a style guide, a design choice, or a technical decision, do so politely and constructively.

---

## Reporting Issues

If someone breaks this code of conduct:

- Stay calm  
- Report the behavior with links and context  
- The repository maintainers will review it in good faith  

There’s no automatic ban or drama — just fairness.

---

## Enforcement

Maintainers of this project reserve the right to:

- Lock a thread  
- Remove comments
- Block abusive users
- Ask folks to rephrase or tone down language

All enforcement is meant to keep this project healthy and usable.

---

Thank you for building with us and respecting the rules.
